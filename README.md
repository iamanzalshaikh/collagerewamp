# collagerewamp

CIMDR Sangli — landing page redesign (Next.js App Router).

**Chintamanrao Institute of Management Development and Research**, Deccan Education Society, Sangli.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Stack

- Next.js, React, TypeScript
- Tailwind CSS
- Lenis (smooth scroll)
